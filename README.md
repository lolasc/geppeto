## Installation

Clone with git:

    git clone https://github.com/lolasc/geppeto.git
    cd geppeto

Install with 
    
    couchapp push . http://localhost:5984/geppeto

or (if you have security turned on)

    couchapp push . http://adminname:adminpass@localhost:5984/geppeto
